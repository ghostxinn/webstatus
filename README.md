## Information
Check a website/hostname status by checking the status code. 
Applies to every website.

## Dependencies 
- Python 3.x
- Any Terminal
- OS : Linux/Windows
- Requests Module

## How To Use? 
```
---> Linux OS <---
python3 webstatus.py 
---> Windows <--- 
python webstatus.py
```
